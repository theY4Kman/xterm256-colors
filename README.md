# xterm256-colors

Easily colorize text printed to an xterm-256color terminal emulator

```python
from xterm256_colors import Fore256, Back256

print(Fore256.CHARTREUSE1('Hello,'), Back256.HOTPINK('World!'))
```

# Installation

pip install xterm256-colors
